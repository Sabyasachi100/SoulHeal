# SoulHeal 🌿

SoulHeal is a comprehensive mental health and wellness platform designed to support students and individuals on their journey to inner peace.

## 🚀 Features

- **Personal Dashboard**: Track your wellness journey, daily streaks, and mood.
*   **Self-Assessments**: Standardized clinical tools (GAD-7, PHQ-9, PSS-10) for anxiety, depression, and stress.
*   **Booking System**: Connect with certified professional counselors for video sessions.
*   **Resource Center**: Access curated articles, audio meditations, and wellness guides.
*   **Live Chat**: Instant support when you need to talk.

## 🛠️ Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Framer Motion (Animations), Lucide React (Icons).
- **Backend**: Node.js, Express.
- **Database**: MongoDB Atlas.
- **Authentication**: JWT (JSON Web Tokens).

## 🏃 Getting Started

### 1. Clone the repository
```bash
git clone <your-repo-url>
cd SoulHeal
```

### 2. Setup Backend
```bash
cd backend
npm install
# Create a .env file with:
# PORT=5000
# MONGO_URI=your_mongodb_uri
# JWT_SECRET=your_secret
npm start
```

### 3. Setup Frontend
```bash
cd frontend
npm install
npm run dev
```
